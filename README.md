# Epoch Converter

A modern, fast, and responsive web application for converting Unix timestamps to human-readable dates and vice versa. Built with Next.js, React, and Tailwind CSS.

## Features

- **Live Epoch Clock**: Displays the current Unix epoch time in real-time.
- **Epoch to Date**: Convert any Unix timestamp (seconds or milliseconds) to a human-readable date and time.
- **Date to Epoch**: Convert human-readable dates into Unix timestamps.
- **Dark/Light Mode**: Full support for system, light, and dark themes.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) and [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
